Mirul MAKER Portfolio
Welcome to the official repository for Mirul MAKER! This is a personal portfolio website designed to showcase various experimental projects, ranging from hardware engineering (Raspberry Pi, ESP32) to web development and street photography.

Apple-Inspired Aesthetics: Features a clean, modern design utilizing "iCloud style" glassmorphism, blur effects, and Apple's system typography.

Specialized Project Sub-pages: Includes dedicated landing pages like the RPi4 Cyberdeck, which features a distinct Matrix-inspired cyberpunk aesthetic.
